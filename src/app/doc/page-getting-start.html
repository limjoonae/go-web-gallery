การเรียกใช้งาน GOS Component

สำหรับ Project JSP (WebSphere)

1. Download หรือ Copy file ดังต่อไปนี้ มาไว้ใน Project

Download all file

  • index.html

    • หากมีหน้าเพจอยู่แล้วสามารถ copy บางส่วนไปเพื่อเรียกใช้ component ได้ (สามารถใช้ได้ทั้งไฟล์ .html หรือ .jsp)


  • systemjs.config.js

  • tsconfig.json

  • main.ts

  • app.module.ts

  • app.component.ts

    • นำไฟล์ต่าง ๆ มาวางดังตัวอย่างในรูป



    2. Import Component ที่ต้องการใน app.module.ts ดังตัวอย่างในรูป



    3. Compile ไฟล์ app.module.ts ใน command window (ในไดเรกทรอรี่ที่ไฟล์อยู่) โดยใช้คำสั่ง 

     tsc app.module.ts

    4. เรียกใช้ Component ที่ต้องการ แล้วสั่ง compile ไฟล์ที่ทำการแก้ไข เช่น แก้ไขไฟล์ app.component.ts ใช้คำสั่ง 

     tsc app.component.ts

    สามารถ compile file ทั้งหมดใน command window (ในไดเรกทรอรี่ที่รวมไฟล์ทั้งหมด) ด้วยคำสั่ง 

     tsc

    5. start server run project
